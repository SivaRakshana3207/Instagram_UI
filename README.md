# 📸 Instagram UI Clone

A responsive Instagram-inspired UI built using **HTML** and **Tailwind CSS**. This project replicates the core layout of Instagram including stories, posts, reels, and bottom navigation.

---

## 🚀 Features

* 📱 Mobile-first responsive design
* 🔝 Header with logo, search bar & notifications
* 📚 Stories section with horizontal scroll
* 🖼️ Post feed (image + video reels)
* ❤️ Like, comment, share, save icons
* 🎬 Reel-style video interaction (tap to unmute)
* 🔽 Fixed bottom navigation bar

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (basic interactions)

---

## ⚙️ How to Run

1. Clone the repository
2. Open the project folder
3. Run using Live Server

---

## ✨ Highlights

* Smooth UI similar to Instagram
* Clean layout using Tailwind utility classes
* Beginner-friendly project for frontend practice
